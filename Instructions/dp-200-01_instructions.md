# DP 200 - データ プラットフォーム ソリューションの実装
# ラボ 1 - データ エンジニアのための Azure

**推定時間**: 60 分

**前提条件**: このラボのケース スタディは既に確認していることを前提としています。

**ラボ ファイル**: このラボのファイルは、_Allfiles\Labfiles\Starter\DP-200.1_ フォルダーにあります。

## ラボの概要

受講者は、レッスンとケース スタディで得られた情報をもとに、AdventureWorks のデジタル変革プロジェクトの成果物を確認します。まず、データ活用の進化がどのように組織に新しい機会をもたらしたかを確認します。また、ビジネス ニーズへの対応に活用できる Azure Data Platform サービスを確認し、データ エンジニアが行うタスクを定義します。最後に、受講者は AdventureWorks のデータ エンジニアリングの成果物を完成させます。

## ラボの目的
  
このラボを完了すると、次のことができるようになります。

1. AdventureWorks の進化するデータの世界を特定する
2. AdventureWorks で使用する Azure Data プラットフォーム サービスを決定する
3. データ エンジニアが実行するタスクを特定する
4. AdventureWorks の Data Engineering の成果物を完成させる

## シナリオ
  
あなたは、デジタル変革プロジェクトに関して AdventureWorks の経営陣と従業員に助言するために、シニア データ エンジニアとして採用されました。  AdventureWorks は、10年以上にわたり、消費者や販売業者に自転車や自転車部品を直接販売してきました。ここ数年、同社は、さまざまな業界が最近のクラウド技術の進展を活用し、よりパーソナライズされた魅力的なサービスを顧客に提供していることを目にしてきました。同社は、自分たちの業界でその動きをリードしたいと考えています。

あなたは、IT 部門と連携してディスカバリー ワークショップを実施し、AdventureWorks と顧客の両方のために使用できるデータ プラットフォーム テクノロジーを明らかにするよう依頼されました。同社は、そのテクノロジによって、さまざまなレベルで顧客に近づきたいと考えています。例えば、商品の購入やサービスの利用時に顧客別にカスタマイズした提案を行ったり、使用している自転車の利用統計情報を提供したりしたいと考えています。いっぽう、データ ウェアハウスに保持している既存のレポート システムを管理するという要件もあります。

最初のステップでは、クラウド テクノロジの現状を評価し、次の分析を実行してプロジェクトを実行できることを確認します。

1. AdventureWorks の中で進化しているデータの世界を明らかにする
2. AdventureWorks で使用する Azure Data Platform サービスを決定する
3. データ エンジニアが実行するタスクを確認する
4. AdventureWorks における Data Engineering の成果物を完成させる
